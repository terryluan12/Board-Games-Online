# Board Games Online

This repository contains documentation and references for the **Board Games Online** project.

## Overview
**Board Games Online** is an open-source platform designed to host and play board games on the web. The goal of the project is to build an application that supports multiple highly customizable board games within a single platform.

The project is currently under development. The first planned game to be supported is **Dogan** (get it… Dog-an).
